# Diagramas de Estado

Esta carpeta contiene los **diagramas de estado** hechos como ejercicios prácticos.
Estos diagramas representan los distintos estados por los que puede pasar un objeto o sistema y las transiciones entre ellos.

## Contenido

- **Ejercicio5** -> Diagrama de estados del protector de pantalla de un ordenador.
- **Ejercicio6** -> Diagrama de los distintos estados por los que puede pasar un registro de un archivo maestro en una base de datos, sometido a las operaciones básicas CRUD.
- **Ejercicio7** -> Diagrama que representa los distintos estados por los que puede pasar un mensaje de correo electrónico.
- **Ejercicio8** -> Diagrama de estados del proceso de inicio de sesión (login) de un usuario en una aplicación web.
- **Ejercicio9** -> Diagrama de los distintos estados por los que puede pasar un préstamo bancario.

### Objetivo

- Comprender y aplicar los diagramas de estado
- Representar estados, eventos y transiciones
- Mejorar el diseño y la documentación mediante UML
