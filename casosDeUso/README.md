# Diagramas de Casos de Uso

Esta carpeta contiene los **diagramas de casos de uso**  hechos como ejercicios prácticos.

## Contenido

- **Ejercicio5** -> Diagrama de casos de uso de una plataforma educativa.
- **Ejercicio6** -> Diagrama de casos de uso de una biblioteca.
- **Ejercicio7** -> Diagrama de casos de uso de una empresa llamada *Pequeños Viajeros*.
- **Ejercicio8** -> Diagrama de casos de uso de una empresa de reparto y pedidos.
- **Ejercicio9** -> Diagrama de casos de uso de la aplicación de un banco.
- **Ejercicio10** -> Diagrama de casos de uso de una clínica veterinaria.

### Objetivo

- Practicar la creación de diagramas de casos de uso
- Identificar actores y funcionalidades de distintos sistemas
- Mejorar la capacidad de análisis y modelado
- Aplicar UML a casos reales
